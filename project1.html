Images and Alternate Text

Alt text refers to invisible description of images and used within an HTML code. Alternative text is very important in a way to identify images which have no text associated them. People using screen reader programs to browse the web cannot see or hear what the image represents unless a short label using the alt attributes code to describe the appearance and function of an image on a page.
Guideline 1.1 - Text Alternatives. This page provides general guidelines on what kinds of text to use in an Alt attributes discusses alternate text requirements for images.

Informative Image : Example

This image conveying an impression or emotion.

This photograph shows a tense football game. It's being used to enhance the impression of some article, for example, that tells the readers about the game.

Code Snippet:

<figure>
<img src="./images/fplayer.jpg" alt="A football player hands the ball off to another player" width="320" height="228">
<figcaption>Player image by <a href="https://pixabay.com/users/keithjj-2328014/">KeithJJa(Pixabay.com)
</figcaption>
</figure>

Functional Image : Example

This is stand-alone icon image that has a function.

In this example, the image follows text within a link to inform users that the link will direct them to the home page. It has the text alternative “YouTube Home Page Link” to convey the meaning of the icon.

Contrast and Color Disability

To provide legible text and images for anyone with low vision or color deficiencies we have to find colors in a good contrast, for example, contrast between content and the background, or foreground and background.

WCAG 2.1 guidelines state that colors should have a text-to-background contrast ratio of 4.5:1 for normal text and 3:1 for large text to meet basic AA requirements or 7:1 for normal text and 4.5:1 for large text to meet the stricter AAA requirements.
