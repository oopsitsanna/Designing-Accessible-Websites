Accessibility Statement for Rabbit Land Website

Rabbit Land strives to create online resource that are accessible to all. We are committed to further reviewing and improving the accessibility of this website. All content is presented as html pages and where documents are provided to download. All images contain alt text descriptions.

Measures to support accessibility

Rabbit Land takes the following measures to ensure accessibility of Rabbit Land website:

  • Include accessibility as part of our mission statement.
  • Include accessibility throughout our internal policies.

Conformance status

The Web Content Accessibility Guidelines (WCAG)Links to an external site. defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. Rabbit Land is partially conformant with WCAG 2.0 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Compatibility with browsers and assistive technology

Rabbit Land is designed to be compatible with the following assistive technologies:

  • Chrome, Firefox, and Edge browsers are keyboard accessible and compatible with JAWS screen readers.

Rabbit Land is not compatible with:

  • Browsers older than 3 major versions.

Technical specifications

Accessibility of Rabbit Land relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • WAI-ARIA
  • CSS
  • Java Script

These technologies are relied upon for conformance with the accessibility standards used.

Limitations and alternatives

Despite our best efforts to ensure accessibility of Rabbit Land, there may be some limitations. Below is a description of known limitations, and potential solutions. Please contact us if you observe an issue not listed below.

Events Calendar plugin used on our website has some links that are missing text. These are empty links that do not provide any user benefit.
